# Candidates

These folders are active working copies of the strongest current lanes.

- `non_ttt_m22_base`: merged March 22, 2026 official non-TTT record base.
- `non_ttt_vrl_gptq`: public PR `#569` non-TTT upside lane.
- `ttt_lora`: public PR `#572` LoRA TTT upside lane.

Each candidate keeps the upstream training script and upstream notes in place so we can run, diff, and adapt them without mutating `records/`.
